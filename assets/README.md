Images and assets for the site

- Place images for pages under `assets/images/`.
- Refer to them from HTML like: `<img src="assets/images/your-image.jpg" alt="...">`.
- Keep file sizes optimized for web (JPEG/WEBP for photos, PNG for transparency).
- Example:
  - `assets/images/meta-hero.jpg` used in `meta-ads.html` (if you add it).
